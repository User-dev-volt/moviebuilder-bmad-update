# Scene Structure Reference

Use this during step-04 (Narrative) and step-06 (Polish) to help filmmakers think about what a scene accomplishes and whether the emotional arc is earned.

---

## Scene Types by Narrative Function

| Type | What It Does | Typical Beat Pattern |
|------|-------------|---------------------|
| **Arrival** | Character enters a new space/situation | Orientation → Discovery → Reaction |
| **Confrontation** | Two forces meet / conflict surfaces | Tension build → Clash point → Aftermath |
| **Revelation** | Something hidden becomes known | Setup → Delivery → Processing |
| **Transition** | Character moves from one state to another | Old state → Disruption → New orientation |
| **Intimate** | Two characters share something private | Approach → Core moment → Withdrawal |
| **Solo / Interior** | One character alone with their situation | Stillness → Action → Consequence |
| **Ritual** | Repeated behavior that carries meaning | Establish the habit → Variation → Meaning |

---

## Emotional Arc Patterns

A scene should open at one emotional temperature and close at a different one. The beats are the journey between them.

### Common Arc Shapes

**Rising tension:**
Opens at: calm / routine / neutral
Closes at: danger / awareness / dread
> Beats: something ordinary → something wrong → the recognition

**Deflation:**
Opens at: hope / momentum / certainty
Closes at: defeat / stillness / grief
> Beats: forward motion → obstacle → the weight of it

**Unexpected warmth:**
Opens at: distance / formality / conflict
Closes at: connection / vulnerability / tenderness
> Beats: guarded interaction → a crack appears → something real

**Resolution (earned):**
Opens at: pain / unresolved tension / burden
Closes at: acceptance / clarity / release
> Beats: the weight being carried → the moment of letting go → after

**Ambiguity:**
Opens at: one reading of the situation
Closes at: a different (or uncertain) reading
> Beats: apparent meaning → complication → reframing

---

## Checking Whether an Arc Is Earned

Ask these questions during polish:

1. **Is the opening emotional state clear from the first beat?** The audience needs to know where they're starting from.

2. **Do the beats move the temperature?** Each beat should shift the emotional state slightly — up, down, or sideways. A flat arc means beats aren't pulling their weight.

3. **Is the closing emotional state different from the opening?** If the character ends where they started, either the scene is transitional (ok if intentional) or the beats need more weight.

4. **Is the arc earned or stated?** The beats should SHOW the change, not describe it. "Elias's hand unclenches" earns the arc. "Elias feels better" states it.

## Scene Length Guidelines

| Shard Count | Scene Length | Best For |
|-------------|-------------|---------|
| 2-3 shards | 10-24 seconds | Punctuation beats, transitions, single moments |
| 4-6 shards | 20-48 seconds | Standard intimate scenes, confrontations |
| 7-10 shards | 35-80 seconds | Complex sequences, emotional arcs requiring space |
| 10+ shards | 50+ seconds | Set pieces, rituals, sustained tension |

**Recommendation:** When in doubt, keep scenes tight (4-6 shards). Tighter scenes = stronger continuity = better shard-generation results.

---

## Linking Scenes to Show Bible Themes

If a Show Bible exists, each scene should serve at least one thematic pillar. During step-04, ask:

- "Which of the Show Bible's thematic pillars does this scene serve?"
- "What does this scene add to the story arc — what changes because this scene exists?"
- "Is there a recurring motif that could appear here?"

A scene that can't answer these questions may not be load-bearing. That's OK — not every scene carries thematic weight — but it's worth knowing.
