# Lens Reference

Reference data for the Visual Architect during camera language creation.

## Standard Focal Lengths and Emotional Effects

| Focal Length | Category | Emotional Effect | Best For |
|-------------|----------|------------------|----------|
| 14mm | Ultra-Wide | Extreme distortion, surreal, disorienting | Dream sequences, horror, establishing vast spaces |
| 18mm | Ultra-Wide | Expansive, vulnerable, environmental | Landscapes, architecture, isolation in space |
| 24mm | Wide | Subject small in world, environmental storytelling | Establishing shots, showing character in context |
| 28mm | Wide | Slightly wider than natural, subtle tension | Street photography feel, documentary style |
| 35mm | Standard-Wide | Most natural wide, slight environmental awareness | Walk-and-talk, casual medium shots |
| 40mm | Standard | Very close to human eye perspective | Neutral observation, documentary |
| 50mm | Standard | Human eye equivalent, natural, honest | Dialogue, natural perspective |
| 65mm | Short Tele | Subtle compression, flattering portraits | Medium close-ups, interviews |
| 85mm | Telephoto | Compression, isolation, intimate/oppressive | Close-ups, portraits, claustrophobic intimacy |
| 100mm | Telephoto | Strong compression, voyeuristic distance | Surveillance feel, observation |
| 135mm | Telephoto | Heavy compression, extreme isolation | Emotional peaks, subject divorced from environment |
| 200mm+ | Super Tele | Extreme compression, flattened depth | Stalking, extreme isolation, abstract |

## Shot Type Conventions

| Shot Type | Typical Focal Length | Frame Coverage |
|-----------|---------------------|----------------|
| Extreme Close-Up (ECU) | 85-135mm | Single feature (eye, hand) |
| Close-Up (CU) | 85mm | Face fills frame |
| Medium Close-Up (MCU) | 50-85mm | Head and shoulders |
| Medium Shot (MS) | 35-50mm | Waist up |
| Medium Wide (MW) | 28-35mm | Knees up |
| Wide Shot (WS) | 24-28mm | Full body in environment |
| Extreme Wide (EWS) | 14-18mm | Vast environment |

## Shot Progression Examples

Named emotional arcs through lens changes:

| Name | Progression | Emotional Arc |
|------|------------|---------------|
| Paranoia Spiral | 50mm → 85mm → 135mm | Normal → claustrophobic → suffocating |
| Revelation | 135mm → 85mm → 35mm | Focused obsession → release → context |
| Isolation Build | 35mm → 50mm → 85mm → 135mm | Comfortable → aware → intimate → trapped |
| Power Shift | Wide Dutch → 85mm level | Chaos → control |
| Dream Collapse | 14mm → 24mm → 50mm | Surreal → grounding → reality |

## Camera Movement Vocabulary

| Movement | Effect | Common Usage |
|----------|--------|-------------|
| Static/Locked | Stability, observation, composure | Dialogue, establishing, tension |
| Dolly (push in) | Increasing intimacy or tension | Approaching revelation, emotional crescendo |
| Dolly (pull out) | Revealing context, abandonment | Loneliness, the bigger picture |
| Pan | Following action, surveying space | Scene transitions, tracking movement |
| Tilt | Revealing scale, power dynamics | Architecture, looking up/down at subject |
| Handheld | Urgency, documentary feel, chaos | Action, panic, authenticity |
| Steadicam | Dreamlike glide, controlled movement | Following subjects, ghostly presence |
| Crane/Jib | God's eye, omniscience, scale | Establishing, endings, transitions |
| Dutch Angle | Psychological unease, world off-kilter | Tension, madness, disorientation |
| Whip Pan | Sudden energy, surprise | Scene transitions, shock |
| Zoom (vs dolly) | Voyeuristic compression, surveillance | Thriller, paranoia, unnatural observation |
