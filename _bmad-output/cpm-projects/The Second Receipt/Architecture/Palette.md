# Color Palette — The Second Receipt

> Extracted from the Cinematic Style Guide. This file is consumed by the Cinematographer agent during shard generation.

## Allowed Colors

| Name | Hex | Usage Context |
|------|-----|---------------|
| Powered-Down Screen | `#0B101A` | All shadows in Cold spaces — sterile void, not mystery |
| Slate Blue | `#2C3E50` | Elias's suit, car, corporate architecture — visual armor, oppressive weight |
| Tablet Blue | `#5B8DD9` | Tablet screen uplight — Elias's cold key source |
| System Green | `#8FBCAA` | Hallway/stairwell fluorescents — the siege light |
| Institutional Bleach | `#F0F4F8` | Legal documents, sterile walls — The Erasure |
| Organic Parchment | `#EDE0C4` | Mara's "whites" — aged lace, piano keys, sheet music, tea saucers |
| Piano Mahogany | `#7B3F2A` | Piano wood, aged wood surfaces — the immovable anchor |
| Tungsten Retreat | `#D4722A` | Practical lamp light — warmth under siege, night scenes |
| Legacy Gold | `#F0C04A` | Golden-hour window — Mara's world in daylight |

## Forbidden Colors

| Color | Reason |
|-------|--------|
| Saturated Crimson (e.g. `#E60000`) | Signals thriller urgency and physical violence — this film moves in slow erosion, not confrontation |
| Pure Black (`#000000`) | Too cinematic and mysterious — Cold shadows belong to the System, not the romantic void |
| Pure White (`#FFFFFF`) | Too clean and heavenly — The Erasure is institutional, not divine |
| Neon / High-Vis Synthetics (e.g. highlighter yellow, electric magenta) | Introduces cyberpunk/pop frequency — light in this film glows or radiates, it never screams from a pigment |
| Easter Pastels (e.g. mint green, baby pink, powder blue) | Communicates weightlessness — this film is about weight, and pastels would undercut the melancholic tension |

## Color Meanings

- **`#0B101A` Powered-Down Screen:** The System at rest — not mystery, not depth, just absence of processing. The dark that doesn't breathe.
- **`#2C3E50` Slate Blue:** Visual armor — a storm cloud that never breaks, all weight, no release. The color of authority that has forgotten it was once chosen by a person.
- **`#5B8DD9` Tablet Blue:** The language of data — cold arithmetic translated into light, the color that turns Mara into a case number.
- **`#8FBCAA` System Green:** The face of infrastructure — the color that makes skin look tired and eyes look empty.
- **`#F0F4F8` Institutional Bleach:** The Erasure — the color of the machinery that legally deletes a life. Violent when placed next to warmth.
- **`#EDE0C4` Organic Parchment:** Time made visible — a white that has been lived in, breathed on, and slowly loved into cream.
- **`#7B3F2A` Piano Mahogany:** The weight of legacy — the immovable anchor, the color of things made to last beyond a single owner.
- **`#D4722A` Tungsten Retreat:** Warmth under siege — the same soul as Legacy Gold, but concentrated, precious, fighting for its corner.
- **`#F0C04A` Legacy Gold:** The soul in full bloom — the color of things grown or made by hand, the world as it was before the contract arrived.
